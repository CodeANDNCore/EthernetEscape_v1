# 🚀 EthernetEscape

EthernetEscape — мощный и легковесный инструмент на C# для комплексной оптимизации сетевого соединения и обхода глубокого анализа пакетов (DPI) в операционных системах Windows.

Основные возможности:

🌐 DNS Optimizer: Быстрая смена DNS-серверов. Встроены пресеты: Google, Cloudflare, Xbox, AdGuard (Anti-Ads), GeoHide и другие.

🚀 Turbo Mode: Автоматическая настройка параметров TCP-стека (Netsh) для снижения задержек и стабилизации коннекта.

📊 Network Health: Встроенная утилита для замера пинга до глобальных узлов.

🔄 Emergency Reset: Кнопка мгновенного сброса всех сетевых настроек Windows до заводских (Winsock/IP reset).


🛠 Технологии:

Language: C# (.NET Framework / .NET)

UI: WinForms (Dark Material Style)

Rights: Требуются права администратора для работы с драйвером и сетевыми интерфейсами.

📦 Установка и использование:
1. Скачайте последний релиз.

2. Запустите EthernetEscape.exe от имени администратора.

3. Выберите нужный DNS
